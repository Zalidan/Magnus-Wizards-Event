# 01 — La vision

Magnus part d'une idée simple : un contenu long contient souvent plusieurs actifs courts déjà présents, mais invisibles.

Le problème n'est pas seulement de couper la vidéo. Le vrai travail consiste à :

1. repérer les moments qui peuvent vivre seuls ;
2. trouver l'angle qui donne envie de regarder ;
3. formuler un titre ou un hook clair ;
4. vérifier que l'extrait sert un objectif business ;
5. préparer la publication sans perdre le contexte.

## Mauvaise approche

> “Découpe-moi 10 Shorts dans cette vidéo.”

Résultat probable : beaucoup d'extraits moyens, peu de stratégie.

## Bonne approche

> “Analyse cette vidéo comme un éditeur. Identifie les passages qui créent de la curiosité, prouvent une méthode, montrent une transformation ou donnent envie d'en savoir plus.”

Résultat attendu : moins d'extraits, mais meilleurs.

## Les 4 rôles d'un agent de clipping

- **Analyste** : comprend le contenu long.
- **Éditeur** : choisit les moments à potentiel.
- **Marketeur** : transforme le passage en promesse regardable.
- **Contrôleur qualité** : refuse les clips faibles.

Magnus orchestre ces rôles. Ce kit t'aide à reproduire une version simple de cette logique.
